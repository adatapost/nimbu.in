X-UA-Compatible

  • Make IE render IE
  • Best in .htaccess

Meta Charset

  • Before

    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">

  • After

    <meta charset="utf-8">

Favicons

  • Just drop into root directory

Fail-safe jQuery

  • Using Google CDN
  • If that fails or if you are developing offline, uses local file.
  • Uses protocol-relative url
  • Works on http and https, avoid seeing prompts in IE

Mobile viewport

  • Default

  • With H5BP Viewport

normalize.css

  • normalize all browsers to present same defaults.
  • Do not reset good browser defaults.
  • corrects bugs and common inconsistencies.

Helper classes

  • Image Replacement: .ir
  • Clearfix: .clearfix
  • Hiding elements only for display: .visuallyhidden
  • Hiding elements from screen readers, while retaining layout: .invisible
  • Hiding elements from screen readers and browsers: .hide

Print Styles

  • Within same stylesheet.
  • Prevents multiple network requests
  • a[href]:after { content: " (" attr(href) ")"; }
    
.ir a:after, a[href^="javascript:"]:after, a[href^="#"]:after { content: ""; }

Modernizr

  • HTML5 Shiv
  • JS Loader: yepnope.js
  • .no-js, .js can be used for tailored solutions.
  • .no-textshadow/.textshadow

.htaccess

  • Proper MIME types for all files
  • Gzip compression for all files
  • Expires headers for good caching
  • Rewrite URLs from www.example.com to example.com
  • Block access to hidden folders or folder indexes
  • Other servers: github.com/paulirish/html5-boilerplate-server-configs

Build Script

  • Automated optimization of resources and files.
  • ant build / ant build dev / ant minify / ant text / ant clean
  • ant css-split
  • JSHint: ant jshint
  • CSSLint: ant csslint
    • Concats and minified JS/CSS
    • Replace CSS/JS references to minified files
    • Compresses images
    • Optional manifest generation
    • Basic html compression, optional HTML minification
