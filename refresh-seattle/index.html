What Is Sass?

  • Syntactically Awesome Stylesheets.
  • Allows variables, mixins, functions in your stylesheet written in Ruby.
  • This is programming and or similar to writing a pseudo-code except with syntax.
  • White-space sensitive, which means you need to learn to use white-space correctly, unlike CSS.
  • Your Sass file is compiled to a CSS file. Compass is a framework that makes it easy to do so.

    When to use SASS/Compass

    • Want to split CSS,but worried about multiple file requests.
    • Maintainable code
    • Reusable snippets
    • that is easy to change when clients want to. Clients want to change themes, colors, or want a website in a day.
    • You have a set of snippets you want to use for every site but only when relevant and not add to CSS bloat.

    Caveats

    • It is only as good as you are.
    • It is only as maintainable as you make it to be.
    • Use only if everyone who is working on the CSS uses Sass as well.
    • if your grasp of CSS is poor, so will your sass be.
    • It is a means to an end and not an end in itself.
